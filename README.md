# HANDWRITTEN DIGITS RECOGNIZER USING NUMPY + IDX2NUMPY (神经网络)
### "Hello world" of neural networks, explained for begginers using math + code.

Using [MNIST dataset]("http://yann.lecun.com/exdb/mnist/"), python, [numpy](https://numpy.org) and [idx2numpy](https://github.com/ivanyu/idx2numpy) I built a neural network that sucesfully recognize handwritten digit numbers with accuracy of ~ 93% (epoch = 400). <br />
Brief step by step explanation:
1. Data handling using idx2numpy,
2. Neuron definition,
3. Utilizing [ReLu](https://pl.wikipedia.org/wiki/ReLU) as a neuron activation function,
4. Utilizing [softmax](https://en.wikipedia.org/wiki/Softmax_function) as output probability determinator,
5. Using $-\ln(x)$ as loss function
